Setup Your Project Folder

You received a ready-to-deploy ZIP from Claude. Extract it and open a terminal inside the hpas-deploy folder. Run these commands:

📁 Your folder should look like this:
hpas-deploy/
  ├── src/
  │   ├── App.jsx  ← your HPAS app
  │   └── main.jsx
  ├── index.html
  ├── package.json
  ├── vite.config.js
  ├── vercel.json
  └── .gitignore
Terminal — install dependencies:
# Open terminal in the hpas-deploy folder, then run:
npm install

# Test it runs locally first:
npm run dev

# Visit http://localhost:5173 — you should see HPAS!
💡 If you don't have Node.js installed, download it first from nodejs.org — get the LTS version.
2

Push to GitHub

Create a new GitHub repository and push your code. You already have a GitHub account — just follow these steps:

2a — Create a new repo on GitHub.com:
🌐 Go to github.com/new → Name it hpas-form-agent → Set to Public → Click "Create repository" → DO NOT add README or .gitignore
2b — Run these commands in your terminal:
# Initialize git in your project folder
git init
git add .
git commit -m "initial: HPAS form agent"

# Connect to your GitHub repo (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/hpas-form-agent.git
git branch -M main
git push -u origin main
🔑 When it asks for a password, use a GitHub Personal Access Token (not your password). Create one at github.com/settings/tokens → "Generate new token (classic)" → check repo scope.
3

Deploy on Vercel (Free)

Vercel gives you a free public URL and auto-deploys every time you push to GitHub. No credit card required.

1️⃣ Go to vercel.com/new and sign up/login with your GitHub account.
2️⃣ Click "Import Project" → find hpas-form-agent → click Import.
3️⃣ Vercel auto-detects Vite. Leave all settings as default. Just click "Deploy".
4️⃣ Wait ~60 seconds. Vercel builds and deploys automatically. Done! 🎉
⚠️ Important: Your Anthropic API key should NOT be hardcoded in the file for a public site. In Vercel dashboard → your project → Settings → Environment Variables → add VITE_ANTHROPIC_KEY = sk-ant-.... Then in App.jsx line 1, change to:
const API_KEY = import.meta.env.VITE_ANTHROPIC_KEY || "";
4

Your Live Link!

After deployment, Vercel gives you a permanent public URL like:

https://hpas-form-agent.vercel.app

Share this link with anyone. It works on all devices — desktop, tablet, mobile.
Every time you push code to GitHub, Vercel automatically re-deploys in under 60 seconds.

To update your site after making changes:
# Make your changes to App.jsx, then:
git add .
git commit -m "update: describe your change"
git push

# Vercel auto-deploys in ~60 seconds. That's it!

Bonus: Custom Domain (Optional)
